##  技术栈
- **框架**: Next.js  + React  + TypeScript  
- **UI 插件库**: Shadcn UI 
- **智能合约**: Foundry
- **密码学层**: Primus SDK (zkTLS/zkFHE)文档：@docs/primus.md
- **样式**: Tailwind CSS  
- **状态管理**: Jotai 
- **Web3**: wagami

## 开发规范
- 使用 TypeScript 进行类型安全的开发
- 采用 Tailwind CSS 进行响应式设计
- 遵循 Next.js 约定的文件结构和路由规则
- 使用 Jotai 进行轻量级状态管理
